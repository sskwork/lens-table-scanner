# 📸 Google Lens Table Scanner

A professional mobile web application that integrates with Google Lens to scan handwritten tables and convert them to digital format. Built as a Progressive Web App (PWA) for seamless cross-platform use.

## ✨ Features

- **Google Lens Integration** - Open Google Lens directly from the app
- **Smart Paste Detection** - Automatically detects tabs, spaces, and commas
- **Dynamic Table Generation** - Creates A, B, C columns with 1, 2, 3 rows
- **Editable Cells** - Make corrections directly in the table
- **Value Extraction** - Get cell values with coordinates (A1, B2, etc.)
- **Excel Export** - Download as CSV for Excel
- **History** - Save and load previous scans
- **Offline Support** - PWA works without internet
- **Mobile Optimized** - Perfect for all screen sizes
